
# Get Started

Supported yapi version

* latest
* 1.9.2
* 1.8.9




## Docker Compose

```shell script
> cd compose
> docker-compose up -d
```

Then visit

> http://127.0.0.1:8080

Login with

name | value
---|---
username | admin@ymfe.org
password | ymfe.org

## Customization

### yapi version

> compose/.env

### yapi config

> compose/yapi/config.json

Change the default admin user name or something else
